# Keyboard Shortcuts

## Draw modes

| Key | Action             |
| --- | ------------------ |
| `S` | Select mode        |
| `L` | Draw Lane          |
| `J` | Draw Junction      |
| `W` | Draw Crosswalk     |
| `T` | Draw Signal        |
| `P` | Draw Stop Sign     |
| `B` | Draw Speed Bump    |
| `K` | Draw Parking Space |
| `A` | Draw Clear Area    |
| `C` | Connect Lanes mode |

## Editing (mapbox-gl-draw)

| Key                    | Action                                         |
| ---------------------- | ---------------------------------------------- |
| `Escape`               | Return to Select mode                          |
| `Delete` / `Backspace` | Delete selected element                        |
| Click segment          | Add vertex at click point (in `direct_select`) |
| Drag vertex            | Move vertex                                    |

## Map navigation

| Action        | Interaction  |
| ------------- | ------------ |
| Pan           | Click + drag |
| Zoom          | Scroll wheel |
| Zoom in / out | `+` / `-`    |
| Reset zoom    | `0`          |

## History

| Key            | Action             |
| -------------- | ------------------ |
| `Ctrl+Z`       | Undo               |
| `Ctrl+Shift+Z` | Redo               |
| `Ctrl+Y`       | Redo (alternative) |
